ReadMe File


-The purpose of this program is to simulate the game plinko.

-When the program starts the user will be asked for the filename 
 of the .txt file.

-The user will also be asked to enter the amount of times they would
 like the simulation to run.

-After that the program will execute by first printing out the game
 board from the .txt file requested.

-Then the program will output the number of columns along with
 the probabilty that the plinko will land into one of those final
 columns.
